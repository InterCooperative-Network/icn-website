Contribute to the InterCooperative Network

ICN is a community-driven effort. We welcome contributors of all backgrounds and skill levels. Whether you write code, design, document, or organize, there's a place for you here.

How to Get Involved

1.

Join the Community on Matrix

The first step is to connect with us on Matrix. Introduce yourself, ask questions, and see what discussions are active.

Join our Matrix Community!

Connect with other members, ask questions, and participate in discussions on our Matrix server.

Join Matrix Chat
2.

Explore Projects & Issues

Browse our list of projects on the /projects page or look through GitHub issues, especially those tagged 'good first issue'.

See Projects
3.

Understand Our Workflow

Familiarize yourself with our general contribution guidelines (CONTRIBUTING.md) and how we use GitHub for pull requests and code reviews.

Read Contribution Guide (Example)
4.

Fork, Code, & Open a Pull Request

Once you've found something to work on, fork the relevant repository, make your changes, and submit a Pull Request. Don't hesitate to ask for help!

ICN GitHub Repositories
5.

Contribute Beyond Code

Not a coder? You can still help! We need writers for documentation, designers, testers, community moderators, and advocates. Join the discussion to find out how.

Join the Discussion
