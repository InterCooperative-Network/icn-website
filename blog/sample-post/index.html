Welcome to the ICN Blog

Posted on July 26, 2024 by The ICN Team

Welcome to the official blog of the InterCooperative Network (ICN). We're launching this space to share development updates, technical insights, and community news.

What is ICN?

ICN is a substrate daemon for the cooperative internet. It provides infrastructure for:

  • Timebanks: Hour-for-hour exchange systems with mutual credit ledgers
  • Mutual Credit: Community currencies backed by mutual trust
  • Democratic Governance: Built-in proposals, voting, and CCL contracts
  • Federation: Networks of cooperatives with maintained local autonomy

Current Status

ICN has completed 20 development phases with 1,134+ tests passing. The system includes:

  • 16 specialized Rust crates
  • DIDs with Ed25519 cryptography
  • Web-of-participation trust graph
  • QUIC/TLS secure networking
  • Mutual credit ledger with Merkle-DAG anchoring
  • Pilot Web UI for cooperative communities

What's Next: Track C1

The next milestone is Track C1: Pilot Deployment. We're seeking cooperative communities interested in running real-world pilots of ICN for timebanking, mutual credit, or cooperative governance.

If you're part of a cooperative, timebank, or community organization interested in piloting ICN, please reach out via our Discord or GitHub Discussions.
