Understanding ICN Architecture

ICN is built as a layered architecture where each layer serves specific functions. Hover over each layer to learn more.

ICN Architecture Stack

Applications & UI

Pilot Web UI icnctl CLI icn-console
User interfaces for timebanking, governance, and administration

Governance

Proposals Voting CCL Contracts
Democratic decision-making with domain-scoped proposals and voting

Economics & Ledger

Mutual Credit Double-Entry Merkle-DAG
Mutual credit ledger with double-entry accounting and DAG anchoring

Runtime & Compute

Actor System Distributed Compute Storage
Tokio-based actor runtime with distributed compute and persistent storage

Networking

QUIC/TLS Gossip Protocol mDNS Discovery
Secure P2P mesh networking with trust-gated gossip and peer discovery

Identity & Trust

DIDs Ed25519 Trust Graph
Decentralized identity with web-of-participation trust and Ed25519 cryptography
